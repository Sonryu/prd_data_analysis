## Projeto

Construa um dashboard que permita receber um arquivo de dados de teste estático, selecionar o intervalo relevante, gerar o gráfico do intervalo selecionado e uma tabela com os dados obtidos na tarefa 8.

O projeto deve conter:

1. Cabeçalho;
2. Seleção de arquivos;
3. Exibição de gráfico;
4. Seleção de intervalo do gráfico;
5. Gráfico do intervalo selecionado;
6. Tabela com dados do intervalo;

O melhor dashboard pode ser elegível a um prêmio.
